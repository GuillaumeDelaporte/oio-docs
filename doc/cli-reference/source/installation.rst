Installation
~~~~~~~~~~~~

#. Install OpenIO repository package:

   .. code-block:: console

      # yum -y install http://mirror.openio.io/pub/repo/openio/sds/15.12/el/openio-sds-release-15.12-1.el.noarch.rpm


#. Install OpenIO python package:

   .. code-block:: console

      # yum -y install python-oiopy
