Support
=======

Updating
========

To update your box, change into your box directory and update it using:

   .. code-block:: console

    # vagrant box update

Remove the OpenIO-SDS instance in VirtualBox: Select your box > Machine > Remove > Remove all files


Then rerun:

   .. code-block:: console

    # vagrant up
