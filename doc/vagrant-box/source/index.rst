===========================
Official OpenIO Vagrant Box
===========================

Abstract
~~~~~~~~

We provide a Vagrant box that can help you using OpenIO Object Storage solution. This box is for TESTING ONLY.
Documentation is available on Atlas_ that hosts the images. Vagrantfiles are available on GitHub_.

It installs a standalone namespace, named _OpenIO_, with OpenStack Swift proxy and OpenStack Keystone identity service, as well as the S3 proxy.
More information about OpenIO Swift Gateway_.
